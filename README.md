# PitchArena AI 🚀

PitchArena AI is an AI chatbot built with Zapier that helps users stress-test startup ideas through multiple perspectives.

## Live Demo
https://pitch-arena-ai.zapier.app

## Features
- Co-Founder Mode
- Investor Mode
- Devil’s Advocate Mode
- Competitor Mode
- Customer Mode
- Dynamic mode switching
- Startup idea critique and refinement

## Problem
Startup ideas often receive only supportive feedback.

PitchArena AI challenges ideas before they are built.

## How It Works
1. User pitches a startup idea
2. Selects a mode
3. AI analyzes the idea from that perspective
4. User can switch modes anytime

## Example
Idea:
AI study planning platform

Investor Mode:
How do you monetize?
What differentiates you from existing tools?

Devil’s Advocate Mode:
What if users prefer free alternatives?
